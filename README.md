# Smart Student QR Security System
Smart Student QR Security System is a web application designed to improve school security using QR code technology.

Students are issued ID cards with QR codes. When scanned, the system instantly displays the student's identity including:

- Name
- Class
- Student photo
- Parent contact

This helps school security officers verify students quickly and prevent unauthorized access.

## Features

QR code scanning using camera  
Student database  
Instant student profile display  
Simple and clean web interface  

## Technologies Used

Python  
Flask  
HTML  
CSS  
JavaScript  

## How To Run

1. Install requirements

pip install -r requirements.txt

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
